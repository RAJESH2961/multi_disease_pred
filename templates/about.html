About This Project

This Heart Disease Prediction System uses machine learning algorithms to predict the possibility of heart disease based on certain user inputs such as age, blood pressure, cholesterol levels, exercise-induced angina, and more.

The goal of this project is to help individuals understand their risk factors for heart disease and seek timely medical advice. It serves as a tool for early diagnosis based on commonly available health parameters.

Technologies Used:

  • Python & Flask (Backend)
  • Bootstrap (Frontend)
  • Machine Learning Algorithms (for prediction)
